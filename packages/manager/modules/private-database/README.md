# manager-private-database

## Install

```sh
yarn add @ovh-ux/manager-private-database
```

## Usage

```js
import 'angular';
import privateDatabase from '@ovh-ux/manager-private-database';

angular
  .module('myApp', [
    privateDatabase,
  ]);
```
